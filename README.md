# TunesKit Audio Capture

## Overview

TunesKit Audio Capture is an audio recording tool designed for Windows users to capture sound from various applications and streaming platforms. It enables users to record audio in high quality while maintaining the original format and metadata. The software supports multiple audio sources and formats, providing flexibility for different recording needs.

## Key Features

**Capture audio from multiple applications simultaneously**  
**Preserve original audio quality and metadata**  
**Support for recording in common audio formats such as MP3, AAC, WAV, and FLAC**  
**Customize audio output settings including sample rate and bit rate**  
**User-friendly interface for straightforward operation**  
**Ability to split and edit recorded audio files**  
**Record streaming music, internet radio, and system sounds**
